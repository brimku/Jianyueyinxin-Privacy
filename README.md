# 简乐音信 隐私政策
最后更新：2026年4月7日

## 1. 信息收集
本应用为本地音乐播放器，不会主动收集用户的任何个人信息，包括但不限于姓名、电话、位置、通讯录、照片、账号密码等。
应用仅在本地读取设备中的音乐文件用于播放，不会上传任何数据到服务器。

## 2. 信息使用
- 不收集、不上传、不存储用户个人数据
- 仅本地使用音乐文件，用于播放功能

## 3. 信息共享
本应用不会向任何第三方共享、转让或披露用户信息。

## 4. 第三方服务
本应用不含第三方广告、统计、分析 SDK，不与任何第三方服务进行数据交互。

## 5. 数据安全
所有操作均在本地完成，无云端存储，数据安全由系统保障。

## 6. 未成年人保护
本应用不面向 13 周岁以下儿童。

## 7. 政策更新
隐私政策若有更新，将在此页面公布。
